Introduction to CSS

This is the first assignment for the Introduction to CSS course. CSS, or Cascading Style Sheets, is a style sheet language used for specifying the presentation and styling of documents written in markup languages, most commonly HTML. It is a fundamental technology of the World Wide Web, working alongside HTML (which provides structure and content) and JavaScript (which adds interactivity). Key aspects of CSS include: Styling Web Content: CSS controls the visual appearance of web pages, including elements like fonts, colors, spacing, layout, and animations. Declarative Language: CSS is a declarative language, meaning you declare what you want the styles to be, and the browser then applies those styles to the selected elements. Cascading Rules: The term "Cascading" refers to the rules that govern how styles are prioritized and applied when multiple styles might conflict. This system ensures a consistent and predictable application of styles across a website.
